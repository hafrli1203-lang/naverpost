# METRICS — 숫자로 증명할 항목

공통: `C:\project\_AGENCY_OS\HARNESS_STANDARD.md`. metrics-auditor가 측정, 메인 작업자가 기록.

## 코드
| 항목 | 목표 | 측정 방법(프로젝트 명령에 맞게) |
|---|---|---|
| build 에러 수 | 0 | 빌드 명령(`package.json` scripts / 파이썬 실행) |
| 타입/컴파일 에러 수 | 0 | typecheck / tsc / 컴파일 |
| lint 에러 수 | 0 | lint 명령 |
| 테스트 pass/fail | fail 0 | 테스트 명령(없으면 "미검증") |
| 추적된 민감 파일 수 | 0 | `git ls-files` 패턴 점검(내용 미열람) |
| 승인 없는 의존성 추가 수 | 0 | package.json/lock·requirements diff |
