# 블로그 파이프라인 v1.0

## 개요
여러 상점의 네이버 블로그 콘텐츠를 생성하고, 본문과 이미지를 검토한 뒤 세션 단위로 저장해 다시 불러올 수 있는 워크플로우다.

## 현재 단계
1. 키워드 선택
2. 본문 확인
3. 이미지 확인
4. 완료

## 현재 범위
- `/api/keywords`로 키워드 옵션 생성
- `/api/article`로 본문 생성 및 검토
- `/api/image/generate`, `/api/image/regenerate`로 이미지 생성
- `/api/sessions`로 세션 저장 및 복원

## 제외된 범위
- 네이버 블로그 임시저장
- 네이버 블로그 발행
- `/api/publish` 기반 외부 게시 연동

## 비고
- 최종 단계는 결과 확인용 UI이며 외부 시스템에 저장하지 않는다.
- 이어서 작업할 수 있도록 localStorage와 세션 저장 기능은 유지한다.
